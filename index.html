Programar a través de lenguajes de programación y líneas de código. Trabajar con otros profesionales de la informática y el diseño para llevar a cabo el proyecto. Crear funcionalidades especificas para la web. Supervisar y mejorar el rendimiento y capacidad de la página web

HTML

(Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) es un lenguaje descriptivo que especifica la estructura de las páginas web

CSS

CSS son las siglas de “Cascading Style Sheets” (hojas de estilo en cascada). CSS es un lenguaje para la composición y estructuración de páginas web (HTML o XML). Este lenguaje contiene elementos de codificación y se compone de estas “Cascading Style Sheets” que también se llaman archivos CSS

JS

JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás

Preguntas frecuentes

En este apartado encontrará las dudas sobre nuestro curso y la respuesta a los mismos.

¿Qué es la programación WEB?

Desarrollo o programación web: define la creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías del lado del servidor y del cliente que conllevan unos procesos de base de datos y el uso de un navegador para realizar determinadas tareas o mostrar información.

¿Comó y cuándo voy a estudiar?

El curso consta de tres componentes: teoría con refuerzo en el simulador, tareas para practicar de forma independiente y escritura de código con un tutor. Puedes estudiar en el simulador cuando te convenga, mientras que las tareas están ligadas a un ciclo de 2 semanas.

¿Comó puedo pagar el curso?

Puedes pagar el importe total del curso en un solo pago utilizando la banca en línea, el pago al contado o el pago en línea. O dividir el pago en varias cuotas.

¿Estas listo para aprender programación WEB?

¡Prepárate para una experiencia interesante! Luego de darte la bienvenida, el primer paso será: Aprender teoría escrita con lenguaje sencillo, ilustrada con muchos casos de la industria con chistes y lifehacks profesionales.
